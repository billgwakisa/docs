## Build with banks, ship like a startup

Bridge lets software teams launch financial features while banks/FSPs retain custody and compliance. We provide standardized adapters, strong governance, and developer‑first APIs.

## Onboarding at a glance

- For businesses: create a sandbox account and share KYB details
- Select the services you need (Embedded Finance, Payments, Open Banking); we issue credentials and permissions
- For FSPs: register your platform, provide API credentials and supported products; we run connectivity checks
- Environments: start in sandbox; move to production after tests and approval
